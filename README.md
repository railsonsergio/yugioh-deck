## :rocket: **Getting Started**

```bash
  # clone the repository
  $ git clone https://github.com/railsonsergio/yugioh-deck

  # enter directory
  $ cd yugioh-deck

  # install dependences
  $ yarn install

  # run the project
  $ expo start
```
